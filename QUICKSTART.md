# Quick Start

## 1. Create Database
```bash
createdb rtb_equipment
```

## 2. Start Backend
```bash
cd backend
npm install
npm run dev
```

## 3. Create Admin (new terminal)
```bash
cd backend
npm run create-admin
```

Login: `admin@rtb.gov.rw` / `admin123`

## 4. Start Frontend (new terminal)
```bash
cd frontend
npm install
npm run dev
```

## 5. Open Browser
`http://localhost:5173`
